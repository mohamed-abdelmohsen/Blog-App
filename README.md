# Blog-App

## Overview
The Blog app allows you to add posts, comment on them and see latest posts.

## Installation
To install the Book Management System, follow these steps:
1. Clone the repository.
2. Open a new terminal and Install project requirments  using `pip install -r requirements.txt`.
3. Configure the database settings in `settings.py`.
4. Run the system using `python manage.py runserver`.

## Usage
Once installed, users can access the system through the web interface at `http://localhost:8000/blog`. From there, they can see posts.

## features
- Users can add posts
- Search posts by tags
- comment on posts
- see latest and most commented posts
- sitemap
